# Кокпит

Персональный веб-дашборд. Заменяет браузерные закладки — объединяет быстрый доступ к ссылкам, командную палитру ⌘K, виджеты Todoist/Notion и FAB-кнопки для быстрого захвата.

## Стек

- **Next.js 15** (App Router) + TypeScript
- **TailwindCSS** + **shadcn/ui**
- **cmdk** — командная палитра
- **fuse.js** — fuzzy-поиск
- **better-sqlite3** — локальная БД (SQLite)
- **node-cron** — фоновая синхронизация

## Быстрый старт

```bash
cp .env.example .env
# Заполнить .env нужными ключами
npm install
npm run dev
```

## Деплой (Docker)

```bash
docker compose up -d
```

## Команды ⌘K

| Команда | Алиасы (RU) | Действие |
|---------|-------------|---------|
| `add <url>` | `добавить`, `+` | Добавить ссылку |
| `note <текст>` | `заметка`, `заметку` | Заметка → Notion |
| `task <текст>` | `задача`, `задачу` | Задача → Todoist |
| `open <имя>` | `открыть` | Переключить пространство |
| `theme` | `тема` | Переключить тему |
| `goto <имя>` | `перейти` | Перейти по ссылке |
